POKESTATS - UI/UX DESIGNER - FRONT-END Developer

This was my first big project as a Front-End Developer where I, along with team, were given the task of building a pokemon app that allowed users to search, filter, and save their favorite Pokémon. Users can create an account with the information stored in a back-end. I was also the UI / UX designer on this app, creating and styling each and every element using Adobe Photoshop, Adobe XD, Affinty Designer, and Vanilla CSS. The gifs were provided by the Pokémon Dev team.

SPECTRE TV - UI/UX DESIGNER - FRONT-END Developer

Spectre TV is an app that allows users to browse, search, and stream their favorite tv shows. Build with React, this app gets data from the TMDB api, YouTube, and a third-party video retriever. This was a personal project to test my skills and knowledge on apis,search and filters, Links, ROutes, and prop drilling in React. I also designed this app myself using just Vanilla CSS from scratch and used Adobe XD for prototyping and mockup design.

SMURFS REDUX - UI/UX DESIGNER - FRONT-END Developer

A school assignment, the goal was to create a Smurf app that allowed users to add their own Smurfs to the 'Smurf Village' from the ground up, using only Redux. I completed the assignment and stretch goals which was to add an update and delete feature that makes changes to the api's update and delete. I also decided to bring this app to life a bit more, visually, but adding in Smurfs from the Smurf movie and adding in some CSS animations for character!

CINEMA142 - UI/UX DESIGNER - FRONT-END Developer

Built with React, Cinema142 is an app that allows users to browse and watch movie trailers, and if they liked the trailer, they are allowed to stream the full movie. This app brings data through TMDB api, the YouTube api, and a third-party video player. I was the UI/UX designer for this app and used Tilt compoent library for the poster hover animations. Vanilla CSS was used to style this app and was build from scratch.
